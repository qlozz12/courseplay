Installation

Installing the released version

Copy the archive file ZZZ_courseplay.zip to your mods folder. That's all, folks!

The mods folder is located at:

  • Windows (7 and above):
  • C:\Users\[USER NAME]\My Documents\MyGames\FarmingSimulator2015\mods
  • Mac OS X (Download/DVD/Steam-Version):
  • /Users/[USER NAME]/Library/Application Support/FarmingSimulator2015
  • Mac OS X (X 10.7 and above):
  • /Users/[USER NAME]/Library/Containers/com.focus-home.farmingsim2015/Data/Library/Application Support/FarmingSimulator2015

Installing the developer version

Unpack the downloaded the zip file to a folder of your choosing. Inside, you will find a folder named courseplay-master. Archive the content of that folder to a new zip file named ZZZ_courseplay.zip. Then proceed installing it as described for the released version (see above).
